The unbeatable tic-tac-toe game was one of my favorite projects! I really enjoyed creating a simple AI opponent that never loses.

I had already tried to write a bunch of rules to make the computer hard to beat, but this time I did it properly: I used the minimax algorithm.

The minimax algorithm tells the computer to look ahead at the possible outcomes of all its moves. It tries not to lose, but it will go for a win if you leave it an opening.

I guess it's kind of boring for the player that they can never win the game, but it was fun for me to program it.

Take a look at the source code on Github if you want to see how I made my unbeatable tic-tac-toe game.
